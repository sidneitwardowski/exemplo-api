# Leitor Arquivos

### Requisitos
* Java 11+

#### Construir
```
mvn clean package
```

#### Executar Windows
```
java -jar target\ExemploApiRest-1.0.0.jar
```

#### Executar Unix
```
java -jar target/ExemploApiRest-1.0.0.jar
```

### Orientações

*  Para acessar a documentação da api, digite http://localhost:8080/api/swagger
